# Weather App
A Python CLI weather app using the OpenWeatherMap API.

## Setup
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/weather-app`
2. Install dependencies: `pip install requests python-dotenv`
3. Create a `.env` file and add: `OPENWEATHER_API_KEY=your_key_here`
4. Run: `python weather.py`
